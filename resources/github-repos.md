# GitHub 开源补充资源

这些项目用于实验和理解系统，不替代中国企业案例。

## 供应链优化

- samirsaci/supply-chain-optimization  
  https://github.com/samirsaci/supply-chain-optimization  
  用途：用线性规划理解工厂选址、固定成本、变动成本、运输成本和需求。

- jckantor/ESTM60203  
  https://github.com/jckantor/ESTM60203  
  用途：运筹学 notebook，补优化建模基础。

## 工厂/物流仿真

- OpenFactoryTwin/ofact  
  https://github.com/OpenFactoryTwin/ofact  
  用途：生产和物流物料流数字孪生，理解工厂流动、订单、资源和扰动。

## ERP/MRP/制造系统

- apache/ofbiz-framework  
  https://github.com/apache/ofbiz-framework  
  用途：开源 ERP/CRM/SCM/MRP/MFG 系统，理解企业流程。

- microsoft/PartsUnlimitedMRP  
  https://github.com/microsoft/PartsUnlimitedMRP  
  用途：制造资源计划训练应用，适合理解订单、物料、集成服务。
