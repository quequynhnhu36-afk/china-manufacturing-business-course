# 整车 BOM 成本结构入门

这不是精确成本表，而是帮助小白理解：一辆车的毛利不是从天上来的，它由产品定义、零部件、供应链和制造效率共同决定。

## 常见成本块

| 成本块 | 说明 | 经营问题 |
| --- | --- | --- |
| 动力电池 | 电芯、模组/电池包、BMS、热管理 | 电池价格下降是否真实传导到毛利？ |
| 电驱系统 | 电机、电控、减速器 | 自研是否降低成本或提升性能？ |
| 车身和底盘 | 白车身、底盘、悬架、制动 | 车型平台化是否降低成本？ |
| 智能座舱 | 芯片、屏幕、座舱域控、软件 | 配置堆叠是否吞噬毛利？ |
| 智能驾驶 | 传感器、计算平台、线束、软件 | 智驾投入能否转化为溢价？ |
| 内外饰 | 座椅、玻璃、灯具、内饰件 | 豪华感和成本是否匹配价格带？ |
| 制造成本 | 人工、折旧、能耗、质检、物流 | 产能利用率是否摊薄固定成本？ |
| 质保/售后 | 质保准备、维修、召回 | 质量风险是否侵蚀毛利和现金？ |

## 判断方法

1. 先看车型定位和价格带。
2. 再看核心配置是否超过价格带承受能力。
3. 再看产能利用率是否足以摊薄折旧和人工。
4. 最后看毛利率变化是否和这些运营证据一致。

## 小白常见误区

- 只看售价，不看配置成本。
- 只看电池降价，不看价格战同步降价。
- 只看自研故事，不看自研是否改善毛利或质量。
- 只看销量，不看产能利用率和售后成本。
