# 中国制造业公开情报源地图

## 工厂与产能

- 环评公示
- 规划验收
- 土地出让
- 建设项目审批
- 工厂新闻
- 招聘岗位

## 财务与经营

- 年报/20-F
- 业绩公告
- 投资者演示
- 电话会纪要
- 问询函/回复函
- 债券募集说明书

## 技术与供应链

- 专利公告
- 供应商招股书
- 客户定点公告
- 质量认证
- 召回公告
- 采购/招投标公告

## 使用规则

同一个判断至少需要两个来源交叉验证。比如“产能爬坡影响毛利”应同时寻找工厂/产能证据和财务/毛利证据。

## 推荐检索词

| 目标 | 检索词 |
| --- | --- |
| 工厂产能 | 公司名 + 环评 / 规划验收 / 施工许可 / 土地出让 |
| 财务质量 | 公司名 + 问询函 / 回复函 / 20-F / 6-K / 年报 |
| 质量问题 | 公司名 + 召回 / 投诉 / 缺陷产品 |
| 组织变化 | 公司名 + 招聘 + 质量 / 制造 / 供应链 / 售后 |
| 供应链 | 公司名 + 供应商 / 定点 / 客户 / 招股书 |

## 中国民营制造业特殊问题

- 地方政府支持可能改善现金流或利润，但持续性要单独验证。
- 集团内部采购和关联交易可能影响真实成本。
- 供应商账期可能改善自身现金流，但增加供应链压力。
- 批发、交付、上牌口径不同，不能混用。
