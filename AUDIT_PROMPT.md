# AI 审计提示词

请审阅这个 GitHub 仓库：

https://github.com/quequynhnhu36-afk/china-manufacturing-business-course

## 审计目标

这是一个“中国民营制造业经营管理完整教材课”仓库。课程面向经营管理小白，目标是训练学习者用“运营带财务”的方法，看懂中国民营制造业业务单元。第一版案例聚焦新势力整车：理想汽车、小米汽车、小鹏、蔚来、零跑。

请不要只评价排版或语言流畅度，要重点审计它是否真的能帮助一个小白建立经营判断力。

## 请重点审计

1. **课程深度是否够**
   - 是否仍然停留在资料清单层面？
   - 是否真正解释了订单、产能、成本、库存、现金、质量、组织执行之间的关系？
   - 是否能从“看公司介绍”升级到“判断业务单元是否健康”？

2. **中国制造业适配度**
   - 是否贴近中国民营制造业，而不是泛 MBA 课程？
   - 新势力整车案例是否能穿透到工厂、供应链、质量、交付、毛利和现金流？
   - 是否避免了国企、政策平台和空泛战略话术？

3. **公开来源质量**
   - source-dossier 是否只堆年报和 IR？
   - 是否充分使用了公开经营情报源：SEC、HKEX、巨潮、环评、规划、土地、专利、招聘、召回、监管问询、行业报告？
   - 每个案例的来源是否足以支撑经营判断？

4. **教材结构**
   - 12周课程是否合理？
   - 8章教材是否从小白可理解的角度逐步推进？
   - 模板是否能让学习者产出一页业务判断？

5. **案例质量**
   - 理想汽车、小米汽车、小鹏、蔚来、零跑的案例是否有明显浅薄、错误或缺口？
   - 哪些案例应该优先加深？
   - 哪些公开来源应该补充？

6. **GitHub 开源安全**
   - 是否包含不该公开的内部信息、账号、群聊、token、密码、内部链接？
   - 是否有版权风险：整篇复制年报、报告、文章、课程内容？
   - 外部项目引用是否需要更明确的许可证说明？
