# 中国民营制造业经营管理完整教材课

这是一套面向经营管理小白的完整教材课。目标不是背 MBA 术语，而是训练一种务实能力：站在职能经理视角，用运营带财务的方法，看懂中国民营制造业业务单元的经营健康度。

第一版聚焦新势力整车：理想汽车、小米汽车、小鹏、蔚来、零跑。后续扩展到动力电池、智能装备、汽车零部件和智能制造服务商。

## 学习主线

每家公司都按同一套 Top 5 经营变量拆解：

1. 客户/订单：谁买、订单是否稳定、客户和渠道是否健康。
2. 产能/交付：工厂、产线、爬坡、瓶颈、交付节奏。
3. 成本/毛利：价格战、车型结构、规模效应、供应商议价。
4. 库存/现金：库存、应收、预付款、资本开支、经营现金流。
5. 质量/组织执行：质量问题、召回、售后、招聘、组织调整。

关键不是背下五个变量，而是看懂它们如何传导：

> 订单结构变化 -> 产能利用率变化 -> 单车成本变化 -> 毛利率变化 -> 现金流变化 -> 下一轮投入能力变化。

## 12周课程

1. 经营管理小白入门：业务单元、经营变量、销量/收入/利润/现金流的区别。
2. 经营变量传导：把订单、产能、成本、库存、现金、质量串成一张因果图。
3. 订单与客户：订单真实性、交付节奏、客户画像、渠道压力。
4. 产能与交付：工厂、班次、产线、瓶颈、爬坡。
5. 成本与毛利：BOM、规模效应、价格战、车型结构、供应商议价。
6. 库存与现金：库存、应收、预付款、资本开支、经营现金流。
7. 质量与组织执行：召回、投诉、售后、生产质量、组织调整。
8. 产业链穿透方法：电池、电驱、智能座舱、车身、热管理如何反推整车经营。
9. 案例实战一：理想、小米、小鹏用同一模板做穿透。
10. 案例实战二：蔚来、零跑用同一模板做穿透。
11. 新势力对比：理想、小米、小鹏、蔚来、零跑的经营变量对比。
12. 一页业务判断：输出可复用的业务单元 memo。

## 学习方法

每周只做一件事：用公开材料回答一个经营问题。不要先看观点文章，先看来源。

推荐顺序：

1. 看教材章节，理解本周变量。
2. 用 `templates/public-source-checklist.md` 找证据。
3. 用 `templates/company-source-dossier.md` 建公司资料卡。
4. 用 `templates/one-page-business-judgment.md` 写一页判断。

## 边界

- 本仓库只使用公开来源，不放内部资料、内部链接、飞书群、账号、token。
- 不构成投资建议。
- 对年报、问询函、环评、招股书只做链接、摘要、学习问题和来源记录，不整篇复制。
- 对 GitHub 开源项目只做链接和实验说明，遵守原仓库许可证。
